let's begin

For starting our work in python first we need to have an editor or IDE. installed in our systems.

IDE stands for Integrated Development Environment. It is a Graphical User Interface (GUI) based software which allows the programmers to write, debug, release, and deploy their code.

If you search you can find a number of editors.

Some of them are:

  • PyDev
  • Rodeo
  • Sublime Text
  • Wing
  • Spyder
  • IDLE
  • Visual Studio Code

You can download these editors. The links are provided in the download section.
